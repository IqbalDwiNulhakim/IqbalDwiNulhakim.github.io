1. Project: DPPGPP Salary Application

Project Name: DPPGPP Salary Application for Indonesian Army Finance Directorate

Internship Duration: October 2023 - December 2023

Organization: Indonesian Army Finance Directorate (Direktorat Keuangan Angkatan Darat)

Role: Web Developer Intern

Project Description

The DPPGPP Salary Application is a web-based platform developed to streamline salary data management for the Indonesian Army personnel. This project aimed to enhance efficiency, security, and accessibility for managing payroll records, ensuring accurate and secure data processing.

Objectives

  • Develop a user-friendly and secure web application for salary processing.
  • Improve efficiency in salary data storage, verification, and approval.
  • Ensure data security through authentication and authorization features.
  • Integrate an automated report generation system for financial officers.

Key Features

  • User Roles & Authentication: Different roles for Users, Verifiers, and Admins with secure login.
  • Salary Data Management: Upload and manage payroll data in Excel format.
  • Automated Data Verification: Verifiers can approve or reject salary records.
  • Salary Summary Reports: Generates reports based on rank, unit, and period.
  • Secure Access Control: Implementation of role-based access restrictions.

Technologies Used

  • Framework: CodeIgniter 3 (MVC Architecture)
  • Frontend: HTML, CSS, JavaScript
  • Backend: PHP (CodeIgniter Framework)
  • Database: MySQL
  • Security Measures: Authentication & Authorization, SQL Injection Prevention

Project Contributions

  • Designed and implemented the database structure for salary records.
  • Developed frontend UI for users to manage salary data efficiently.
  • Implemented CRUD operations for salary record management.
  • Integrated an authentication system to ensure secure access.
  • Conducted testing and debugging to enhance system performance.

Project Impact

  • Improved salary data processing time and reduced manual errors.
  • Strengthened data security through secure authentication mechanisms.
  • Provided a seamless and efficient experience for military finance officers.
